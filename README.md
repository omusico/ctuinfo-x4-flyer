# 這是一個為四軸而建立的以ZF2實作的內容管理網站
## 目錄結構
- config 設定檔
- docs 說明文件
- module 模組
- vendor 第三方套件
- public 靜態文件資料夾

## 模組
- 前端(Frontoffice)
- 後端(Backoffice)
- 成員(User)
- 課程(Course)
- 內容管理
- 記綠
- 訊息(發佈介接 FB Plurk
- 權限(ACL

## 版本需求
- PHP5.4 up
- Apache 2.2.22 up

## 開發環境建立
cd /var/www/YOUR-INSTALLATION-PATH
composer update
